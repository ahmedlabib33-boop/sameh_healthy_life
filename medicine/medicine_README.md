# تشغيل التقرير التفاعلي

الملف `app.py` تطبيق Streamlit مستقل بذاته (البيانات مضمّنة داخله، لا حاجة لملفات إضافية).

## التشغيل محليًا
```bash
pip install -r requirements.txt
streamlit run app.py
```

سيفتح المتصفح تلقائيًا على `http://localhost:8501`.

## المحتوى
- الملخص التنفيذي
- المنهجية وحدود الاستدلال
- مصفوفة الأدلة الغذائية (عناصر مثبتة الفائدة + عناصر واعدة غير حاسمة)
- تقييم جودة الأدلة لكل عنصر
- مقارنة الطب النبوي بالأدلة الحديثة
- جدول التداخلات الدوائية العملي
- فجوات المعرفة والخلاصة النهائية

كل الجداول قابلة للبحث والتصفية مباشرة داخل التطبيق.
